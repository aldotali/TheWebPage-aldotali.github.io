Itchy Feet

ItchyFeet is our Senior Design Project for the degree completion. Our group is made of 4 people and it is an ongoing project. ItchyFeet is an Android based route optimizer application that minimizes the effort required to creating a traveling trip plan when being a tourist in a new city. ItchyFeet addresses the problems of tedious internet crawling for your perfect route activity by making use of real time data to give personalized recommendations on places to visit in an optimized route map plan. The recommendations consist of touristic nodes, museums, restaurants and possible events that are filtered based on the userâ€™s previous journeys.

XOX Mania

This project was a FPGA project coded using SystemVerilog. It required an 8X8 RGB display. The logic behind the project was simple. On the RGB the user had three choices (these choices could be selected by using Basys3 FPGA) , each of which represented a game type. The user could play a normal TicTac Toe, an SOS game or a TicTacToe against an AI (computer). Special thanks go to my group mate Faaiz Ul Haque student at Bilkent University for giving a large help in the design and the implementation of the project.
